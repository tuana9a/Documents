# how to run

set the environment variable include PORT and ADDRESS by

`export PORT=80`

`export ADDRESS=127.0.0.1`

then

`java jar docs-1.0.0.jar`

or simple specify it in command line

`java -jar docs-1.0.0.jar --server.port=PORT --server.address=BIND`
