CMD (no need IDE)

javadoc để đọc tài liệu java

javac để biên dịch sang byte-code
thông dịch (dịch từng dòng) từ .java sang .CLASS
javac tên_file.java thì tên file không phân biệt hoa thường

java để biên dịch sang mã máy
java tên_class thì tên file có phân biệt hoa thường
biên dịch (dịch tất) từ .class sang mã máy
chạy file "a.class" gõ "java a" còn "java a.class" không được

QUY TẮC ĐẶT TÊN

tên file và tên class phải giống nhau mới chạy được
nhưng thử đặt tên file khác vẫn chạy được
